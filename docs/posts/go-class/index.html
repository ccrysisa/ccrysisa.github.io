Hello world!

编写经典的 hello world 程序，在 Go Playground 上执行，介绍了该简单程序的组成部分，例如 package, import, func 等 (main 函数必须放置在 main 包里)。

根据 用户文档 安装 Go 并通过 go run 执行 hello world 程序 (这里我安装的是 Go 语言此时的最新版 1.22.1):

# install Go
$ rm -rf /usr/local/go && tar -C /usr/local -xzf go1.22.1.linux-amd64.tar.gz

# add the line below into ~/.bashrc if you use bash as your shell
export PATH=$PATH:/usr/local/go/bin

$ source ~/.bashrc
$ go version
go version go1.22.1 linux/amd64

$ mkdir hello && cd hello
# write code about print hello world in hello.go and config in go.mod
$ go run .
Hello, world!
