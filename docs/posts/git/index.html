Git 常见问题及解决

git pull/push 遇到 Port 22 connect timeout

网络问题导致 22 端口被禁止，无法正常使用 ssh。切换成 443 端口并且编写配置文件即可：

1
2
3
4
5

$ vim ~/.ssh/config
# In ~/.ssh/config
Host github.com
HostName ssh.github.com
Port 443
